# Contributing

## Contributing Rules:
When contributing, please ensure to follow these rules and guidelines:
- Localize any text when adding any text associating with the mod's content.
- Any localization should belong in their respective file under the Localization folder.
- If you have a bug report and want to fix it, feel free to do so. Please ensure to test it fully afterwards.
- If you want to do cleanups and formatting fixes, you are free to do so.
- Add any code comments and documentation if truly necessary.
- If you want to submit a sprite or a different asset for something, ask Astra or Zee.

This list will be updated later if needed.
